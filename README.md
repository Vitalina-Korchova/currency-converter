# Currency Converter

**Currency Converter** is a modern Next.js 16+ web app for real-time currency conversion and historical rate charts.

## 🚀 Tech Stack
- Next.js 16+ & React 18
- TypeScript
- Tailwind CSS
- Recharts (charts)
- Redux Toolkit (state & history)
- GSAP (animations)
- gh-pages (GitHub deployment)

## ⚡ Features
- Real-time currency conversion (USD, EUR, UAH, etc.)
- Historical rate charts
- Stores last 10 conversions in localStorage
- Responsive design for mobile & desktop
- Smooth animated charts and UI
